Why these packages are needed
1. Streamlit

Streamlit allows you to build an interactive web app for your heart attack prediction model.

It provides simple functions like st.title(), st.button(), and st.write() to display content.

2. Pandas

Used to read CSV/Excel datasets and handle structured data.

Example: pd.read_csv('data.csv') or manipulating data for model input.

3. NumPy

Provides efficient numerical operations on arrays.

Example: converting user input into a NumPy array for prediction.

4. Matplotlib & Seaborn

matplotlib is a general plotting library.

seaborn builds on matplotlib and makes statistical visualizations easier.

Example: plotting correlation heatmaps of features.

5. Plotly

For interactive visualizations like dynamic graphs in the web app.

Example: plotting heart rate vs. age or cholesterol with tooltips.

6. Scikit-learn

Required if you are training models or doing preprocessing (scaling, encoding, splitting data).

Example: RandomForestClassifier, StandardScaler, train_test_split.

7. Pickle5

For loading pre-trained models saved as .pkl files.

Example: model = pickle.load(open("heart_model.pkl","rb")).
